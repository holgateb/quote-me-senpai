About us

Quote Me, Senpai is a fun anime knowledge quiz and random anime quote generator created as Project 1 by the Group 1 students in UW BootCamp who are anime fans. It is a muti choice quiz game made by using AnimeChan API where the player has to choose one correct answer from four given options. If a correct answer is slected the player will see a positive animation quote pulled from Nekos Best API and will continue towards next round of quiz by collecting streaks, whereas if the wrong answer is selected they'll see a negative GIF amd quote. Along with the quiz we have also created a function of a random quote page which can be easily accessed by clicking Random Quote Generator.
